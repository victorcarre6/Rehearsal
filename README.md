# Outil de révision interactif

Application web locale qui vous permet de réviser des questions/réponses organisées par thème.
L'interface minimaliste vous aide à vous concentrer sur l'apprentissage sans distraction.

<img width="1577" height="813" alt="image" src="https://github.com/user-attachments/assets/98f4232b-6421-4426-baa2-43351d8e471a" />

## Installation et utilisation

```bash
cd path/to/rehearsal
./install.sh

rehearsal
```

Par défaut l'application utilise le port 8080.
Pour utiliser un port différent :

```bash
export REHEARSAL_PORT=3000
rehearsal
```
